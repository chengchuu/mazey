addStyle

  • addStyle(style: string, options?: { id?: undefined | string }): boolean
    • Defined in index.ts:1559

    EN: Add <style> in <head>.

    ZH: 添加样式标签; style: 样式标签内的字符串; id: <style> 标签的 id; 返回: 添加成功/失败.

    Example 1: Add the <style> with id, and repeated invoking will update the content instead of adding a new one.

    addStyle(
  `
    body {
      background-color: #333;
    }
  `,
  {
    id: 'test',
  }
);
// <style id="test">
//   body {
//     background-color: #333;
//   }
// </style>

    Example 2: Add the <style> without id, and repeated invoking will adding a new one.

    addStyle(
  `
    body {
      background-color: #333;
    }
  `
);
// <style>
//   body {
//     background-color: #333;
//   }
// </style>

    Parameters

    • style: string

      Style string.

    • Default value options: { id?: undefined | string } = { id: '' }
      • Optional id?: undefined | string

        id in <style>

    Returns boolean

    Success/Fail
